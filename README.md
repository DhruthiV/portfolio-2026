# Portfolio

A personal portfolio site built as a single-page bento-grid dashboard, with a
separate Projects page backed by a live, read-only Notion database.

## Stack

- React + TypeScript + Vite
- Tailwind CSS v4 (`@tailwindcss/vite` plugin)
- shadcn/ui conventions + custom theme (`src/theme.css`)
- `motion` (Framer Motion's successor package) for transitions
- `react-router-dom` for client-side routing
- Vercel serverless function as a Notion API proxy
- Deployed on Vercel
